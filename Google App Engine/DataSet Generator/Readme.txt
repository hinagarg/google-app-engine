1. Run and Compile Workload Generator Source code as below:

> Creat class files for the java files and run class file:
javac WorkloadGenerator.java
java WorkloadGenerator (run class file)

> Create jar file out of the class file: 
jar -cf jar-file-workloadgenerator.jar  WorkloadGenerator.Class
> Run jar file
java -cp jar-file-workloadgenerator.jar WorkloadGenerator
